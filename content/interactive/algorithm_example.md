# Алгоритм: Поиск максимального элемента в массиве

## Шаги алгоритма:
1. Принять, что первый элемент массива — текущий максимум.
2. Для каждого следующего элемента массива:
3.   Сравнить его с текущим максимумом.
4.   Если элемент больше текущего максимума, сделать его новым максимумом.
5. После проверки всех элементов выдать текущий максимум.

## Контрольный вопрос:
Дан массив: [8, 3, 12, 5, 9]
Какое число будет результатом работы алгоритма?
A) 3
B) 8
C) 12
D) 9

**Правильный ответ: C) 12**
