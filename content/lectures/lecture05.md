# Лекция 5 — Массивы (структуры данных)

**Тема:** Как хранить много данных
**Цель урока:** Познакомиться с массивами — простейшей структурой данных
**Время:** 20 минут
**Ключевые понятия:** массив, элемент массива, индекс, размер массива

## 1) Введение

Представьте, что вам нужно записать оценки 30 учеников. Создавать 30 отдельных переменных (оценка1, оценка2, оценка3...) — неудобно. Для таких случаев существуют **массивы**.

**Массив** — это структура данных, которая хранит набор элементов одного типа. Представьте массив как пронумерованные ячейки (почтовые ящики), где в каждой ячейке лежит одно значение.

## 2) Основная часть

### Как устроен массив?

У массива есть:
- **Имя** — например, "оценки"
- **Элементы** — сами значения
- **Индексы (номера)** — чтобы обратиться к нужному элементу
Массив "оценки":
┌─────┬─────┬─────┬─────┬─────┐
│ 5 │ 4 │ 3 │ 5 │ 4 │
└─────┴─────┴─────┴─────┴─────┘
0 1 2 3 4 ← индексы

**Важно!** В программировании счет начинается с 0, а не с 1. То есть:
- первый элемент имеет индекс 0
- второй элемент — индекс 1
- третий элемент — индекс 2
- и так далее

### Как работать с массивами?

**Обращение к элементу:**
оценки[0] = 5 (первая оценка)
оценки[1] = 4 (вторая оценка)

**Запись в массив:**
оценки[2] = 3 (третья оценка стала равна 3)

**Чтение из массива:**
вывести оценки[3] (выведет 5)

### Цикл по массиву

Очень часто массив обрабатывают с помощью цикла:
НАЧАЛО
объявить массив оценок: [5, 4, 3, 5, 4]
сумма = 0
ДЛЯ i ОТ 0 ДО 4
сумма = сумма + оценки[i]
КОНЕЦ ЦИКЛА
среднее = сумма / 5
вывести среднее
КОНЕЦ
Этот алгоритм считает средний балл.

### Примеры массивов в жизни

1. **Список покупок:** ["хлеб", "молоко", "яйца", "сыр"]
2. **Температура за неделю:** [18, 20, 19, 22, 21, 19, 18]
3. **Номера автобусов:** [12, 15, 23, 45, 67]

## 3) Пример/практика

**Задание 1:**
Дан массив чисел: [10, 20, 30, 40, 50]
Напишите алгоритм, который найдет сумму всех элементов.

**Решение:**
НАЧАЛО
массив = [10, 20, 30, 40, 50]
сумма = 0
ДЛЯ i ОТ 0 ДО 4
сумма = сумма + массив[i]
КОНЕЦ ЦИКЛА
вывести "Сумма = " + сумма
КОНЕЦ

**Задание 2:**
Дан массив оценок: [4, 5, 3, 5, 4, 3, 5]
Посчитайте, сколько пятерок получил ученик.
