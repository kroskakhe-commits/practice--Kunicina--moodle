1. Массив — структура данных, хранящая набор элементов одного типа, идентифицируемых по индексу.
2. Список — структура данных, где каждый элемент содержит ссылку на следующий.
3. Стек — структура данных LIFO (последним пришел — первым вышел).
4. Очередь — структура данных FIFO (первым пришел — первым вышел).
5. Сортировка пузырьком — простой алгоритм сортировки.
6. Бинарный поиск — поиск элемента делением массива пополам.
7. Линейный поиск — последовательный просмотр всех элементов.
8. Рекурсия — вызов функцией самой себя.
9. Факториал — произведение чисел от 1 до n.
10. Сложность алгоритма — мера времени/памяти.
11. Граф — вершины и ребра.
12. Дерево — иерархическая структура.
13. Хеш-таблица — поиск по ключу.
14. Блок-схема — графическое представление алгоритма.
15. Псевдокод — запись алгоритма словами.
16. Исполнитель — кто выполняет алгоритм.
17. Детерминированность — одинаковый результат.
18. Дискретность — состоит из шагов.
19. Результативность — приводит к результату.
20. Массовость — для многих задач.
21. Индекс — номер элемента в массиве.
22. Длина массива — количество элементов.
23. Сортировка — упорядочивание элементов.
24. Поиск — нахождение элемента.
25. Ветвление — выбор действия по условию.
